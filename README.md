# 🐈 Glitch Cat: Kinetic World

A surreal, atmospheric 2D Canvas animation featuring a walking cat traversing an ever-changing landscape of glitchy buildings, white-bush trees, and shifting weather cycles.

## ✨ Features
- **Dynamic Weather System**: Cycles through clear firefly nights, cloudy mists, and white-noise rain.
- **Procedural Architecture**: Buildings are constructed from kinetic text strings ("VOID", "NULL", "PROTO").
- **Kinetic Road**: A rippling, reactive ground layer that interacts with the cat's movement.
- **Atmospheric Visuals**: Moon phase cycles and twinkling star fields.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Watch the cat wander through the glitch.

## 🛠️ Technical Details
- **Language**: JavaScript (HTML5 Canvas API)
- **Styling**: Minimalist Black & White (Moonlight palette)
- **Performance**: Uses `requestAnimationFrame` for smooth 60FPS rendering.
